# Методы Network imopenlines

---

## imopenlines.network.join

**Scope:** `imopenlines`  
**Кто может выполнять:** любой пользователь  
**Параметры:** `CODE` (обязательный)

**Описание:**  
Подключает к текущему Битрикс24 внешнюю открытую линию с другого Битрикс24.

**Параметры метода:**

| Название | Тип | Описание |
|----------|-----|----------|
| `CODE` | string | Код открытой линии (строка из 32 символов). Можно найти в карточке подключенной линии на странице Контакт-центра Битрикс24 Network |

**Пример запроса:**
```bash
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"CODE": "ab515f5d85a8b844d484f6ea75a2e494"}' \
  https://your-domain.bitrix24.ru/rest/1/webhook_key/imopenlines.network.join.json
```

**Пример ответа (HTTP 200):**
```json
{
    "result": 603,
    "time": {...}
}
```

**Возвращаемые данные:**

| Название | Тип | Описание |
|----------|-----|----------|
| `result` | integer | Идентификатор network-бота подключенной линии (если линия уже подключена — возвращается существующий ID) |
| `time` | time | Информация о времени выполнения запроса |

**Обработка ошибок:**

| Статус | Код | Описание |
|--------|-----|----------|
| 400 | `CODE` | Некорректный код в параметре `CODE` (ожидается строка из 32 символов) |
| 400 | `IMBOT_ERROR` | Не установлен модуль `imbot` |
| 400 | `LINE_NOT_FOUND` | Открытая линия не найдена |
| 400 | `INACTIVE` | Открытая линия в данный момент недоступна |

Общие ошибки: см. [Обработка ошибок](../common/errors.md)

## imopenlines.network.message.add

**Scope:** `imopenlines`  
**Кто может выполнять:** любой пользователь  
**Параметры:** `CODE`, `USER_ID`, `MESSAGE` (обязательные), `ATTACH`, `KEYBOARD`, `URL_PREVIEW` (опциональные)

**Описание:**  
Отправляет сообщение пользователю от имени открытой линии, подключенной в Битрикс24 Network.

> **Ограничения:**
> - Метод недоступен при сессионной авторизации (ошибка `WRONG_AUTH_TYPE`)
> - Не более 1 сообщения каждому пользователю в неделю (для NFR-лицензий ограничений нет)
> - Клавиатура поддерживает только кнопки-ссылки на сторонние сайты

**Параметры метода:**

| Название | Тип | Описание |
|----------|-----|----------|
| `CODE` | string | Код открытой линии (строка из 32 символов) |
| `USER_ID` | integer | Идентификатор получателя сообщения |
| `MESSAGE` | string | Текст сообщения (поддерживает форматирование) |
| `ATTACH` | object | Вложение |
| `KEYBOARD` | object | Клавиатура (только кнопки-ссылки) |
| `URL_PREVIEW` | char | Предпросмотр ссылок: `Y` (по умолчанию), `N` — отключен |

**Пример запроса:**
```bash
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "CODE": "ab515f5d85a8b844d484f6ea75a2e494",
    "USER_ID": 2,
    "MESSAGE": "Подготовили материалы по подключению",
    "URL_PREVIEW": "N"
  }' \
  https://your-domain.bitrix24.ru/rest/1/webhook_key/imopenlines.network.message.add.json
```

**Пример ответа (HTTP 200):**
```json
{
    "result": true,
    "time": {...}
}
```

**Возвращаемые данные:**

| Название | Тип | Описание |
|----------|-----|----------|
| `result` | boolean | `true`, если сообщение отправлено |
| `time` | time | Информация о времени выполнения запроса |

**Обработка ошибок:**

| Статус | Код | Описание |
|--------|-----|----------|
| 403 | `WRONG_AUTH_TYPE` | Метод вызван с сессионной авторизацией |
| 400 | `CODE` | Некорректный код в параметре `CODE` |
| 400 | `IMBOT_ERROR` | Не установлен модуль `imbot` |
| 400 | `NOT_FOUND` | Открытая линия не найдена |
| 400 | `USER_ID_EMPTY` | Не указан идентификатор пользователя |
| 400 | `USER_MESSAGE_LIMIT` | Превышен лимит сообщений (1 в неделю) |
| 400 | `MESSAGE_EMPTY` | Не указан текст сообщения |
| 400 | `ATTACH_OVERSIZE` | Превышен максимальный размер вложения (30 Кб) |
| 400 | `ATTACH_ERROR` | Некорректный объект вложения |
| 400 | `KEYBOARD_ERROR` | Некорректный объект клавиатуры |
| 400 | `WRONG_REQUEST` | Сообщение не отправлено |

Общие ошибки: см. [Обработка ошибок](../common/errors.md)
