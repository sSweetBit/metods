# Методы сообщений imopenlines

---

## imopenlines.message.session.start

**Scope:** `imopenlines`  
**Кто может выполнять:** любой пользователь с правами на диалог  
**Параметры:** `CHAT_ID`, `MESSAGE_ID` (оба обязательные)

**Описание:**  
Запускает новую сессию и переносит в нее указанное сообщение (начать новый диалог на основании сообщения).

**Параметры метода:**

| Название | Тип | Описание |
|----------|-----|----------|
| `CHAT_ID` | integer | Идентификатор чата открытой линии |
| `MESSAGE_ID` | integer | Идентификатор сообщения внутри чата (можно получить через `imopenlines.session.history.get`) |

**Пример запроса:**
```bash
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"CHAT_ID":2043,"MESSAGE_ID":18971}' \
  https://your-domain.bitrix24.ru/rest/1/webhook_key/imopenlines.message.session.start.json
```

**Пример ответа (HTTP 200):**
```json
{
    "result": true,
    "time": {...}
}
```

**Возвращаемые данные:**

| Название | Тип | Описание |
|----------|-----|----------|
| `result` | boolean | `true`, если новая сессия запущена на основании сообщения |
| `time` | time | Информация о времени выполнения запроса |

**Обработка ошибок:**

| Статус | Код | Описание |
|--------|-----|----------|
| 400 | `CHAT_ID` | Указан не корректный идентификатор чата |
| 400 | `CHAT_TYPE` | Указанный чат не является открытой линией |
| 400 | `ACCESS_DENIED` | Недостаточно прав на диалог |
| 400 | `USER_ID` | Не определен пользователь |

Общие ошибки: см. [Обработка ошибок](../common/errors.md)

## imopenlines.message.quick.save

**Scope:** `imopenlines`  
**Кто может выполнять:** администратор или оператор открытой линии  
**Параметры:** `CHAT_ID`, `MESSAGE_ID` (оба обязательные)

**Описание:**  
Сохраняет сообщение из чата открытой линии в список быстрых ответов.

**Параметры метода:**

| Название | Тип | Описание |
|----------|-----|----------|
| `CHAT_ID` | integer | Идентификатор чата открытой линии |
| `MESSAGE_ID` | integer | Идентификатор сообщения для сохранения в быстрые ответы |

**Пример запроса:**
```bash
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"CHAT_ID":1331,"MESSAGE_ID":83507}' \
  https://your-domain.bitrix24.ru/rest/1/webhook_key/imopenlines.message.quick.save.json
```

**Пример ответа (HTTP 200):**
```json
{
    "result": true,
    "time": {...}
}
```

**Возвращаемые данные:**

| Название | Тип | Описание |
|----------|-----|----------|
| `result` | boolean | `true`, если сообщение успешно сохранено в быстрые ответы |
| `time` | time | Информация о времени выполнения запроса |

**Обработка ошибок:**

| Статус | Код | Описание |
|--------|-----|----------|
| 400 | `CHAT_ID` | Указан некорректный идентификатор чата |
| 400 | `CHAT_TYPE` | Указанный чат не является открытой линией |
| 400 | `CANT_SAVE_QUICK_ANSWER` | Ошибка сохранения быстрого ответа |
| 400 | `ACCESS_DENIED` | Недостаточно прав для доступа к чату |

Общие ошибки: см. [Обработка ошибок](../common/errors.md)
