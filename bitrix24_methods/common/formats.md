# Форматы запросов и ответов Bitrix24 REST API

## Формат запросов

Все методы вызываются через HTTPS с использованием протокола REST.

### Базовый URL

```
https://{your_bitrix24_address}/rest/{user_id}/{webhook}/{method_name}
```

### Заголовки

```
Content-Type: application/json
Accept: application/json
```

## Формат ответа

Стандартный ответ содержит:

- `result` — объект с данными результата
- `time` — информация о времени выполнения запроса

### Пример ответа

```json
{
    "result": {...},
    "time": {
        "start": 1773660116,
        "finish": 1773660116.269681,
        "duration": 0.2696809768676758,
        "processing": 0,
        "date_start": "2026-03-16T14:21:56+03:00",
        "date_finish": "2026-03-16T14:21:56+03:00",
        "operating_reset_at": 1773660716,
        "operating": 0
    }
}
```

## Типы данных

| Тип | Описание |
|-----|----------|
| `integer` | Целое число |
| `string` | Строка |
| `boolean` | Булево значение (`true`/`false`) |
| `object` | Объект (словарь) |
| `array` | Массив |
| `datetime` | Дата и время в формате ISO 8601 (RFC3339) |
| `time` | Объект информации о времени выполнения |
| `char` | Символьное значение (обычно `Y`/`N`) |
